# Westward
## About
Westward is a reimplementation of the original Oregon Trail game with modern features.
Westward is written in Java with a C++ rendering backend. 
It features a completely data-driven JSON-based registry for adding or modifying content.
It provides modern rendering techniques such as immediate mode rendering and dynamic FPS to provide a smooth, streamlined experience.

## Requirements
- x86_64 (Windows, Linux) or aarch64 (MacOS)
- 250MB of RAM available
- OpenAL capable GPU/CPU
- OpenGL 2 capable GPU
- 20MB storage available
